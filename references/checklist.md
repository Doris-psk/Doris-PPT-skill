# 质量检查清单（Checklist）

## 🔴 P0 · 一定不能犯的错

- [ ] 所有幻灯片使用同一套主题色，不中途换色
- [ ] 字体不超过 2 种（中英文各一种）
- [ ] 图片比例统一（16:9 / 16:10 / 4:3 三选一）
- [ ] 标题层级清晰：封面 > 章节 > 页面标题 > 正文
- [ ] 没有文字溢出或被裁切

## 🟡 P1 · 重要

- [ ] 每页只讲一个核心观点
- [ ] 正文不小于 14pt（投影可读性）
- [ ] 数据/数字有可视化表达（大字报/图表/卡片）
- [ ] 留白充足，不拥挤
- [ ] 对比色使用有目的性（不是随便用）

## 🟢 P2 · 锦上添花

- [ ] 封面有视觉冲击力
- [ ] 章节过渡页有节奏感
- [ ] 图文搭配合理
- [ ] 卡片式布局整齐

## ⚪ P3 · 细节

- [ ] 页码/底部信息完整
- [ ] 项目符号统一
- [ ] 颜色对比度满足可读性
