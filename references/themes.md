# 主题色预设（Themes）

10 套精心设计的 PPT 主题色板，覆盖杂志风、瑞士风、深色科技风三大体系。
**不允许自定义颜色**——色彩搭配错了画面瞬间变丑，只从以下预设中挑选。

---

## 📖 使用方式

每个主题定义为一个 JS 对象，在脚本中直接引入：

```js
const THEMES = {
  ink_classic: {
    name: "墨水经典",
    bg: "F1EFEA",        // 纸张底色（暖米白）
    fg: "0A0A0B",        // 墨色（纯黑）
    cardBg: "E8E5DE",    // 卡片底色
    accent1: "0A0A0B",   // 强调色1
    accent2: "6B5B4F",   // 强调色2（暖灰）
    accent3: "8B7D6B",   // 强调色3
    textGray: "6B5B4F",  // 灰色文字
    isDark: false,       // 是否为深色背景
  },
  // ...更多主题
};
```

| 新闻 / 工业 / 活力 | 🟠 安全橙 |

## 五、⚠️ 设计原则

1. **一份 deck 只用一套主题**，不要中途换色
2. **杂志风**：衬线标题 + 暖色纸底 + 克制留白
3. **瑞士风**：无衬线 + 纯白底 + 单一锚点色高亮 + 直角
4. **深色科技风**：深色底 + 荧光强调色 + 卡片叠加
5. **不允许混搭**（例如背景用墨水经典、强调色用克莱因蓝）——会彻底违和
6. **不允许用户随便给 hex 值**——展示预设让选
