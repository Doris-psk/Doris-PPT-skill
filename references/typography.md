# 排版系统（Typography）

## 字体选择

| 场景 | 推荐字体 | 备选 |
|------|---------|------|
| 中文标题 | `Microsoft YaHei` / `微软雅黑` | `SimHei` / `Noto Sans SC` |
| 中文正文 | `Microsoft YaHei` | `SimSun` |
| 英文标题 | `Arial` / `Helvetica` | `Calibri` |
| 英文正文 | `Arial` | `Calibri` / `Segoe UI` |
| 等宽（数据） | `Consolas` | `Courier New` |

## 字号层级

| 层级 | 字号 | 用途 |
|:----:|:----:|------|
| Hero | 48-60pt | 封面主标题、大字报数字 |
| H1 | 36-44pt | 章节标题 |
| H2 | 28-32pt | 页面主标题 |
| H3 | 20-24pt | 卡片标题、副标题 |
| Body | 14-18pt | 正文内容 |
| Small | 10-13pt | 说明文字、标注、元数据 |

## 字重原则

- **越大越细，越小越粗**（瑞士风原则）
  - Hero 字重用 Light/Bold — 视风格而定
  - 正文用 Regular (400)
  - 小字用 Medium/Semibold (500-600) 保证可读

## 杂志风排版铁律

1. 标题用粗体（Bold），正文用常规（Regular）
2. 保证行距充足：正文行距 1.5-2 倍字号
3. 关键数据/数字加大加粗突出
4. 同一页面字体种类不超过 2 种
5. 中文和英文混排时，英文用西文字体、中文用中文字体
